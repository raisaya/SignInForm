# Sign-In Form

A modern and responsive Sign-In Form built using HTML** and CSS.  

This project demonstrates:
- Clean and responsive design.
- Use of CSS gradients, shadows, and blur effects.
- Form validation with required fields.

Perfect for beginners to showcase frontend skills or include in a personal portfolio.

## Features
- Responsive layout for mobile and desktop.
- Interactive input focus effects.
- "Remember me" checkbox and "Forgot password?" link.

## How to Use
1. Clone or download this repository.
2. Open `index.html` in your browser.
